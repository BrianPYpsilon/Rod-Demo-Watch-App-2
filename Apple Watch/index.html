Apple Watch Plugin Demo

The Apple Watch plugin allows you to add an Apple Watch extension to your Phone app. The UI is entirely constructed with JavaScript, so you don't need to touch native code at all.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS - App package.

App pages

A page is shown when the user taps a glance, or launches the app itself on the Watch.
